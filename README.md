# 📊 Analisi Dati Ecommerce con SQL e Python

## 📌 Panoramica

Questo progetto consiste nell'analisi di un dataset ecommerce utilizzando SQL e Python.
L'obiettivo è estrarre insight utili sul comportamento dei clienti, le vendite e l’andamento del business.

Il progetto simula un caso reale di analisi dati in ambito aziendale.

---

## 🛠️ Strumenti Utilizzati

* **SQL (SQLite)** → per interrogare e analizzare i dati
* **Python (pandas, matplotlib)** → per elaborazione e visualizzazione
* **Jupyter Notebook** → per l’analisi interattiva

---

## 📂 Dataset

* File: `data.csv`
* Il dataset contiene:

  * ID ordine (InvoiceNo)
  * Codice prodotto (StockCode)
  * Descrizione prodotto
  * Quantità acquistata
  * Data ordine (InvoiceDate)
  * Prezzo unitario (UnitPrice)
  * ID cliente (CustomerID)
  * Paese (Country)

---

## 📊 Analisi Effettuate

### 💰 KPI principali

* Calcolo del fatturato totale
* Calcolo del valore medio per ordine (Average Order Value)

### 📈 Analisi temporale

* Andamento del fatturato mese per mese

### 👤 Analisi clienti

* Identificazione dei clienti con maggiore spesa

---

## 📈 Visualizzazioni

È stato realizzato un grafico che mostra:

* l’andamento del fatturato nel tempo (trend mensile)

---

## 🧠 Insight Principali

* Una piccola percentuale di clienti genera gran parte del fatturato
* Le vendite mostrano variazioni nel tempo, suggerendo possibili trend stagionali
* I dati possono essere utilizzati per migliorare strategie di marketing e vendita
